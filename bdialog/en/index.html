A can be multi-layer nested, highly customizable useful dialog plugin

bDialog have Modal, Alert, Mask and Toast types dialog basis on Bootstrap Modal

Download Feedback

Bootstrap

Can use bDialog in bootstrap framework immediately

Pass params to new dialog

Dialog caller can pass the params to new dialog

Multi-layer modal nested

Can open multi layer nested modal dialog

Attention Although bDialog through hard work achieved multi-layer modal nested, especial solved the need support dame ie8 difficult problem，but suggested that the interface in the interactive design, as far as possible not to use

Customization

By set initialization options, can custom dialog style, function, callback and so on

Themes quick apply

If you want to uses different themes, just need define the css content and set the class name to dialog

Auto adjust position

When the browser window size changes, dialog will reposition to browser window center

Independent context

Dialog content usually uses a remote link, can also be html elements

Rich callbacks

There have rich callbacks, like before, after dialog open, before, after close dialog and so on

Animation

When dialog open, outside of dialog click, reposition, dialog will have animation effect
