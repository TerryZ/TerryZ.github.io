data 插件列表的数据源

  • 类型：object | string
  • 默认值：无默认值，该参数为必填参数
  • 使用方式：

    数据源可以是json数据格式或URL

    json数据格式
    [{id:1,name:'张三',sex:'男'},{id:2,name:'李四',sex:'男'}]

    如上所例数据，showField可设置为name或sex，keyField可设置为id

    url
    http://xxx.com/load

    使用URL执行ajax读取服务端数据，返回数据格式与上述json数据格式一致

showField 显示在列表中的数据字段

  • 类型：string
  • 默认值：'name'
  • 使用方式：
    设置的值为data数据源中的数据字段名

keyField 数据代码列

  • 类型：string
  • 默认值：'id'
  • 使用方式：
    设置的值为data数据源中的数据字段名

lang 插件语言，默认为'cn'简体中文

  • 类型：string
  • 默认值：'cn'
  • 使用方式：
    可设置的语言
    代码 语言 描述
    de German 德语
    en English 英文
    cn Chinese 中文
    es Spanish 西班牙语
    pt-br Brazilian Portuguese 巴西葡萄牙语
    ja Japanese 日语

searchField 查询关键字字段

  • 类型：string
  • 默认值：无
  • 使用方式：
    查询字段，仅为使用URL(ajax)方式查询服务端时，设置后端查询的字段，不设置则默认使用showField设置的字段

multiple 多选模式

  • 类型：boolean
  • 默认值：false
  • 使用方式：
    设置插件选择模式，默认为单选模式

focusDropList 是否在输入框获得焦点时，展开下拉窗口

  • 类型：boolean
  • 默认值：true
  • 使用方式：
    设置当输入框获得焦点时，是否展开数据列表

autoSelectFirst 是否自动选择列表中的第一项内容

  • 类型：boolean
  • 默认值：true
  • 使用方式：
    是否自动选择列表中的第一项内容(输入关键字查询模式，直接使用鼠标下拉并不触发)

autoFillResult 是否自动填充内容

  • 类型：boolean
  • 默认值：true
  • 使用方式：
    若有列表项目被高亮显示，在焦点离开控件后，自动设置该项为选中内容

noResultClean 是否清空输入关键字

  • 类型：boolean
  • 默认值：true
  • 使用方式：
    在输入框中输入内容进行查询，但没有匹配的内容返回，在焦点离开控件后，自动清空输入框输入的内容

formatItem 列表项目显示内容格式化

  • 类型：function
  • 默认值：无
  • 使用方式：
    参数1：data，行数据object格式 返回：string

pageSize 每页显示记录数

  • 类型：number
  • 默认值：10

initField 初始化数据列

  • 类型：string
  • 默认值：'data-init'
  • 使用方式：

    初始化数据列，默认设置为data-init属性，该属性设置了初始化插件时默认显示的内容

    它必须设置在input标签内，例：< input type="text" data-init="20" >

params 查询参数

  • 类型：function
  • 默认值：无
  • 使用方式：

    使用URL进行AJAX查询时，可传递查询参数

    返回结果格式：{'name':'aa','sex':1}

    例如：params : function(){return {'name':'aa','sex':1};}

callback 事件回调，响应项目被选中后的事件处理

  • 类型：function
  • 默认值：无
  • 使用方式：
    参数：data：选中行的原始JSON数据

    callback : function(data){ $('xxx').val(data.name + data.sex); }
