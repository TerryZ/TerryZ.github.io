SelectPage

A simple style and powerful selection jQuery plugin,
including ajax remote data, autocomplete, pagination, tags, i18n and keyboard navigation features

jQuery plugin

Developed based on jQuery 1.x version

Simple style

Simple and refreshing style, adaptive to any UI environment

Remote data

The datasource of list support local json and remote data (ajax)

Customization

Config options to custom SelectPage functions and look

Autocomplete

Input keywords to filter items of all data

Pagination

Showing pagination to indicate a series of related content exists across multiple pages

Tags input

Tags form for mutltiple items selection

Keyboard navigate

Using keyboard to quick navigate, turning page, pick item

I18n

Provide Chinese, English, Japanese, German, Spanish, Brazilian Portuguese languages
