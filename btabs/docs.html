loginUrl 登录跳转的链接，用于处理登录超时校验后跳转

  • 类型：string
  • 默认值：'/'
  • 使用方式：

    loginUrl : 'http://xxx.com/login'

resize 插件初始化后，对框架区域的高度、宽度等内容进行初始化的回调

  • 类型：function
  • 默认值：无
  • 使用方式：

    resize : function(){ $('box').height(100); }

className 自定义样式

  • 类型：string
  • 默认值：无
  • 使用方式：

    className : 'newTheme'

sortable 是否允许标签被拖拽排序，默认允许

  • 类型：boolean
  • 默认值：true
  • 使用方式：

    sortable : false

$.fn.bTabsAdd 打开新的标签页

  • 类型：function
  • 默认值：无
  • 使用方式：

    $('#mainFrameTabs').bTabsAdd(123,'用户管理','http://aaa.com/user');

$.fn.bTabsClose 关闭已打开的标签页

  • 类型：function
  • 默认值：无
  • 使用方式：

    $('#mainFrameTabs').bTabsClose(123);
